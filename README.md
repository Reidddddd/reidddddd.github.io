## 了缘

万物有往有来。你在这里有所得，心里动了一念，这便是缘起了。缘起之后，需要有一个安放它的去处，让它自然地结束——这便是了缘。

了缘不是交易，也不是义务。它是一项古老而朴素的理解：当一段际遇给予了你什么，你以某种方式回应它，这段际遇便圆满地完成了自己。回应过了，你的心不再牵挂，它可以继续向前。没回应，那念缘气便还悬在那里，偶尔在你心里轻轻晃一下。

所以这里设一个了缘的去处，是给那份心念一个落点。你觉得够了、不需要，那便是够了。你觉得想给一个回响，这里便是。
